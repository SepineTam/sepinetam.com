# Projects

## Stata-MCP

*Founder & Developer (2025)*

An open-source Python-based tool that enables LLMs to write and execute Stata code automatically. Aims to build the foundation for future automated economic research agents.

[GitHub Repository](https://github.com/sepinetam/Stata-MCP)

## DeepEcon

*Developer (2025)*

An open-source Python package for double machine learning with GPU acceleration. Optimized for CUDA/MPS with a clean API similar to EconML.

## NBER-MCP & AER-MCP

*Developer (2025)*

Knowledge retrieval tools for top economics journals. Enables automated literature search through LLMs, serving as the knowledge base for future research agents.

## DtaDock

*Developer (2025)*

An automated data management platform based on Flask, providing data management for Stata-MCP. Also serves as the underlying data management tool for future EconResearchAgent.

## BeyesV

*CTO & Developer (2025)*

An AI-powered accessibility solution based on YOLO and SmolVLM to assist visually impaired individuals. Provides navigation assistance and environmental awareness for the blind.

## Gresh - Yangtze River Delta Green Development

*Team Member (2024)*

Explored green development in the Yangtze River Delta region from the perspective of integrated development. Analyzed the applicability of the Environmental Kuznets Curve in China and created ESG reports for selected companies. Established WeChat official account "Gresh-绿意庇护" and website gresh.com.cn to raise environmental awareness.

Won Third Prize at Shanghai University Zhixing Cup.
